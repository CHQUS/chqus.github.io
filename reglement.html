Règlement de l'école


Le règlement de l'école CHQUS est conçu pour assurer un environnement d'apprentissage sûr, respectueux et productif pour tous les élèves et le personnel.

Ce règlement comprend des directives concernant le comportement des élèves, les règles de sécurité, les politiques anti-harcèlement, etc.

  • Cliquez ici pour consulter le règlement


    • Modalités d'inscription

    Pour vous inscrire à la formation CHQUS, veuillez suivre les étapes suivantes :

    1. Remplissez le formulaire d'inscription en ligne sur notre site web.
    2. Payez les frais d'inscription selon les instructions fournies.
    3. Assistez à la séance d'orientation obligatoire.
    4. Commencez votre formation selon le calendrier établi.

    Pour plus d'informations sur les modalités d'inscription, veuillez nous contacter via le formulaire de contact.
